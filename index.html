Projects

GitMe is a comprehensive full-stack React web application that combines the capabilities of GPT-4 and GitHub APIs. The app offers users concise and informative summaries of their Github repositories, assisting fellow programmers with a tool that can be used to professionally showcase their work on their resumes. During a hackathon, me and my team showcased this project, resulting in a remarkable placement of 3rd out of over 250 participating teams.

ImageGuru is a robust image classification model developed entirely in Python By leveraging the CIFAR-10 dataset and a diverse set of tools and libraries including Matplotlib, scikit-learn, Pandas, Tensorflow, NumPy, and Keras, I meticulously optimized the model's accuracy and conducted in-depth analysis on error data. The project was presented to a panel of judges, where it earned the highest raw accuracy score among almost 100 cohort members.
